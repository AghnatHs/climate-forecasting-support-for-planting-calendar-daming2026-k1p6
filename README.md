# Climate Forecasting as Support for Planting Calendar

## Project

1. Use historical weather observations from 1940 to 2025.
2. Build a forecasting model for future weather conditions.
3. Convert predicted weather into planting suitability decisions.
4. Support planting calendar recommendations, with rice as the reference crop.

## Repository Structure

- `main.ipynb` - main analysis notebook
- `dataset/` - weather datasets used in the project
- `literatures(open-access)/` - reference papers and supporting literature

## Notebook Content as of now

- library installation and imports
- dataset loading
- WMO weather code normalization
- missing data analysis
- descriptive statistics
- distribution plots for numeric, datetime, and categorical variables
